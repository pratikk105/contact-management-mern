# Contact Management Web App (MERN)

A small web application built for an interview task to demonstrate MERN stack fundamentals.

## Features
- **Frontend**: Built with React.js using Vite.
- **Backend**: Node.js and Express.js API.
- **Database**: MongoDB Atlas (Cloud).
- **Functionality**: Contact form with validation and a live-updating contact list.

## How to Run Locally
1. **Backend**: Navigate to `/server`, run `npm install`, then `node index.js`.
2. **Frontend**: Navigate to `/client`, run `npm install`, then `npm run dev`.
